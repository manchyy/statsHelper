# Mu Online Stats Helper

Depending on your class, stats given upon reset and the number of resets, the script calculates your total points and distributes them based on formulas that I found work the best.

Please keep in mind that it's nowhere close to being finished.

## Getting Started

### Installing

* Download the latest AHK version
* Run the statsHelper.ahk
